# User guide — how users operate the platform

End-user documentation for the mandated flows (constitution Art. I, DR-0003; spec FR-073). The printable guide lives here; in-app contextual help content is versioned seeded data (FR-072) — the two must agree.

Updated in the closure commit of any story that changes user-facing behaviour. Baseline assembled by HAP-21; per-flow pages are drafted by the UI story that ships the flow:

| Page (drafted by) | Audience / flow |
|---|---|
| `completing-your-assessment.md` (HAP-8) | Individual — monthly self-assessment |
| `moderating-your-team.md` (HAP-9) | Manager — review, divergence, carry-forward |
| `bu-dashboard.md` (HAP-11) | BU Lead — reading rollups, suppression, completion |
| `initiative-register.md` (HAP-13/14) | Manager+ — registering and updating initiatives |
| `bu-declarations-and-metrics.md` (HAP-15) | EVP/delegate — weekly declaration, monthly metrics |
| `harris-submission.md` (HAP-17) | EVP — generating, reviewing, exporting the submission |
